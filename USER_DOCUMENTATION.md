# ZenaManage User Guide

## Welcome to ZenaManage
ZenaManage is a comprehensive multi-tenant project management system designed to help teams collaborate, manage projects, and track progress efficiently.

## Getting Started

### 1. Login
- Navigate to the login page
- Enter your email and password
- Click "Sign In"

### 2. Dashboard Overview
The dashboard provides a comprehensive overview of your projects and team activities:

#### KPI Cards
- **Active Projects**: Number of ongoing projects
- **Tasks Completed**: Total completed tasks
- **Team Members**: Number of active team members
- **Documents**: Total documents in the system

#### Recent Projects
- View your most recent projects
- See progress bars and status indicators
- Quick access to project details

#### Upcoming Tasks
- Tasks due soon
- Priority indicators
- Direct links to task details

#### Quick Actions
- **New Project**: Create a new project
- **New Task**: Add a new task
- **Invite Member**: Invite team members
- **Upload Document**: Upload project documents

## Project Management

### Creating a Project
1. Click "New Project" from the dashboard or projects page
2. Fill in project details:
   - Project name
   - Description
   - Start date
   - Due date
   - Priority level
   - Team members
3. Click "Create Project"

### Managing Projects
- **View Details**: Click on any project to see full details
- **Edit Project**: Modify project information
- **Manage Tasks**: Add and organize project tasks
- **Manage Team**: Assign team members to projects
- **Archive Project**: Move completed projects to archive

### Project Status
- **Planning**: Project in planning phase
- **In Progress**: Active development
- **On Hold**: Temporarily paused
- **Completed**: Finished project

## Task Management

### Creating Tasks
1. Navigate to the Tasks page
2. Click "New Task"
3. Fill in task details:
   - Task title
   - Description
   - Project assignment
   - Assignee
   - Priority level
   - Due date
4. Click "Create Task"

### Task Status
- **To Do**: Not started
- **In Progress**: Currently being worked on
- **Review**: Under review
- **Completed**: Finished

### Task Priorities
- **High**: Urgent tasks requiring immediate attention
- **Medium**: Important tasks with normal priority
- **Low**: Tasks that can be completed when time permits

### Bulk Actions
- Select multiple tasks using checkboxes
- Choose bulk action from dropdown:
  - Mark as completed
  - Change priority
  - Reassign tasks
  - Delete tasks

## Team Management

### Inviting Team Members
1. Click "Invite Member" from dashboard
2. Enter member email address
3. Select role (Project Manager, Member, Client)
4. Click "Send Invitation"

### Team Roles
- **Project Manager**: Full access to projects and team management
- **Member**: Can create and manage tasks, view projects
- **Client**: Read-only access to assigned projects

### Managing Team
- View all team members
- Edit member roles and permissions
- Remove team members
- View member activity

## Document Management

### Uploading Documents
1. Click "Upload Document" from dashboard
2. Select file(s) to upload
3. Choose project assignment
4. Add description and tags
5. Click "Upload"

### Document Organization
- **By Project**: Documents organized by project
- **By Type**: Filter by document type
- **By Date**: Sort by upload date
- **By Author**: Filter by uploader

### Document Actions
- **View**: Open document in browser
- **Download**: Download to local device
- **Edit**: Modify document details
- **Share**: Share with team members
- **Delete**: Remove document
