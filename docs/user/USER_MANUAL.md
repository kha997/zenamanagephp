## üîß TROUBLESHOOTING

### Common Issues

#### Login Problems
**Issue**: Cannot log in to the system
**Solutions**:
1. Check username and password
2. Ensure Caps Lock is off
3. Clear browser cache and cookies
4. Try different browser
5. Contact system administrator

#### Performance Issues
**Issue**: System running slowly
**Solutions**:
1. Check internet connection
2. Close unnecessary browser tabs
3. Clear browser cache
4. Disable browser extensions
5. Try different browser

#### File Upload Issues
**Issue**: Cannot upload files
**Solutions**:
1. Check file size limits
2. Verify file type is supported
3. Check internet connection
4. Try smaller file size
5. Contact support if issue persists

## ‚ùì FAQ

### General Questions

**Q: How do I reset my password?**
A: Click "Forgot Password" on the login page and follow the instructions sent to your email.

**Q: Can I access the system from mobile devices?**
A: Yes, ZenaManage is fully responsive and works on mobile devices.

**Q: How do I change my email address?**
A: Go to Settings > Profile and update your email address. You'll need to verify the new email.

**Q: Is my data secure?**
A: Yes, we use enterprise-grade security including encryption, secure connections, and regular backups.

### Project Management Questions

**Q: How many projects can I create?**
A: The number depends on your subscription plan. Contact your administrator for details.

**Q: Can I duplicate a project?**
A: Yes, use the "Duplicate Project" option in the project menu.

**Q: How do I archive completed projects?**
A: Change the project status to "Completed" and it will be moved to the archived section.

**Q: Can I export project data?**
A: Yes, use the Export option in the project dashboard to download project data.

### Task Management Questions

**Q: How do I assign multiple people to a task?**
A: Currently, tasks can only be assigned to one person, but you can add collaborators in the task comments.

**Q: Can I set recurring tasks?**
A: Yes, use the "Recurring Task" option when creating a task.

**Q: How do I track time spent on tasks?**
A: Use the time tracking feature in the task details to log hours worked.

**Q: Can I attach files to tasks?**
A: Yes, use the "Attach Files" option in the task details.

### Technical Questions

**Q: What file types can I upload?**
A: Most common file types are supported including PDF, DOC, XLS, PPT, images, and more.

**Q: How long are files stored?**
A: Files are stored indefinitely unless manually deleted or the project is archived.

**Q: Can I integrate with other tools?**
A: Yes, ZenaManage supports various integrations. Contact your administrator for available options.

**Q: How do I report bugs?**
A: Use the "Report Issue" option in the Help menu or contact your system administrator.
